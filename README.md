# 🧪 SauceDemo Automation - TestNG

This project contains automated test scripts for the SauceDemo website using **Selenium WebDriver** and **TestNG**.

---

## 📌 Features

- Login functionality testing
- Product selection and validation
- Add to cart functionality
- Checkout process automation

---

## 🛠️ Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- Eclipse IDE

---

## 📂 Project Structure

src/
 ├── main/java
 ├── test/java
testng.xml
pom.xml

---

## ▶️ How to Run

1. Right-click on `testng.xml`
2. Select **Run As → TestNG Suite**
